### 💬 Zalo Clone – Real-Time Chat Application
> *Jan 2026 – Present*

<p>
  <img src="https://img.shields.io/badge/Spring%20Boot-3.4.2-6DB33F?style=flat-square&logo=spring-boot" />
  <img src="https://img.shields.io/badge/Java-21-ED8B00?style=flat-square&logo=openjdk" />
  <img src="https://img.shields.io/badge/WebSocket-STOMP-010101?style=flat-square&logo=socket.io" />
  <img src="https://img.shields.io/badge/Spring%20AI-Groq-6DB33F?style=flat-square&logo=spring" />
  <img src="https://img.shields.io/badge/AWS-S3-FF9900?style=flat-square&logo=amazon-aws" />
  <img src="https://img.shields.io/badge/MySQL-8.0-4479A1?style=flat-square&logo=mysql" />
  <img src="https://img.shields.io/badge/Redis-Cache-DC382D?style=flat-square&logo=redis" />
  <img src="https://img.shields.io/badge/Docker-Compose-2496ED?style=flat-square&logo=docker" />
</p>

<table>
<tr>
<td width="50%" valign="top">

**⚡ Real-Time & Messaging**
- Dual-channel WebSocket: `/topic` broadcasts + per-user STOMP queues
- Full message lifecycle: `SENT → DELIVERED → SEEN` with real-time broadcast
- Emoji reactions via single batch query (zero N+1 on paginated history)
- Sender-only recall with instant receiver notification

</td>
<td width="50%" valign="top">

**🔐 Security & Auth**
- JWT HS256 with embedded `tokenVersion` claim
- Instant remote logout across **all** devices on new login
- Custom `ChannelInterceptor` validates JWT on every WebSocket CONNECT
- Spring Security RBAC with protected Admin endpoints

</td>
</tr>
<tr>
<td width="50%" valign="top">

**🤖 AI Features (Spring AI + Groq)**
- Personal AI assistant with 20-message rolling context window
- `@AI` group bot responding asynchronously to mentions (last 30 msgs as context)
- Smart Reply: LLM suggests 3 contextual replies as structured JSON array
- Graceful rate-limit fallback handling

</td>
<td width="50%" valign="top">

**☁️ AWS S3 Media Pipeline**
- Private bucket accessed exclusively via presigned URLs (30-min TTL)
- UUID-keyed storage with MIME-type allowlist (img/video/audio/PDF/Office/zip)
- 50MB upload limit
- HTTP Range request support → native in-browser video/audio streaming

</td>
</tr>
</table>
